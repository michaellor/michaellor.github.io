Git It Got It

This is an app used to search for user data (via the github api) when a github username is provided and submitted into a form. The app returns the name, email, number of repositories the user has, as well as a sampled list of repository names and descriptions.

This app was developed in Javascript, with the use of NPM, Gulp, SASS, and Bower technologies.

To Do List

This is a basic to-do list app. This application serves as a platform for the user to fill a form and add a task, select a checkbox to filter through tasks based on their done status, and to mark a task as done. These tasks are accomplished via components and piping.

This app was developed in Angular, with the use of NPM, Gulp, and Bower.

Triangle Tracker

This is an app that determines if a triangle is a scalene, isosceles, or equilateral. This is achieved with a form that takes 3 numeric user inputs and evaluates what type of triangle the 3 values create. Invalid inputs also trigger alarm messages for incorrect inputs.

This app utilizes Javascript and jQuery.

Github Repository

Queen Attack

This is a module that takes in user input for chess board coordinates of a Queen chess piece and a target chess peice. The module then calculates whether the Queen can destroy the target or not. There is also form validation to check and see if coordinate inputs are of the correct type and within the right parameters.

This app utilizes PHP for module development in a Drupal CMS environment.

Shoe Store Project

This project allows for the addition of Shoe Stores and Shoe Brands. A database is used to create, store, and manage data for the Store class, the Brand class, and a join class between the two. Along with adding stores and brands, stores can also be deleted or updated with a new name. The join class allows for many to many relationships, which ties data between both classes, thus allowing the cross referencing of data between the two classes.

This project uses PHPUnit for unit testing. The project uses PHP, Twig, Silex, MySQL, and Composer.
